# Jikan Anime App

## Assumptions Made
- API data may not have all fields (episodes, score, trailer)
- Only top anime and detail screens implemented

## Features Implemented
- MVVM + Hilt architecture
- Compose UI for Home & Detail screens
- LazyColumn for anime list
- Detail screen shows: poster, title, synopsis, genres, episodes, score
- Trailer playback (if available)
- Clickable links to trailers
- Tried Room database caching 

## Known Limitations
- Trailer playback may not work for some videos
- No offline support for detail screen
